# 🚀 Featured Engineering Projects

---

### ⛓️ ChainSeal (Academic Credential Verifier)
A decentralized academic credential verification web application designed to eliminate transcript tampering.
*   **Tech Stack:** Solidity, Hardhat, Ganache, React.js, Node.js
*   **Core Features:** Smart contract-driven verification, cryptographic proof of records, and an intuitive institution dashboard.

---

### 🔍 NetForensics (Packet Monitoring & Detection System)
A Python-based packet monitoring and network analysis tool designed to capture raw traffic and parse files for forensic reporting.
*   **Tech Stack:** Python, Scapy, Streamlit
*   **Core Features:** Live packet sniffing, automated threat pattern matching, and an interactive web-based dashboard for log visualization.

---

### 💊 MediMart (Pharmacy Management System)
A robust, beginner-friendly management system tailored for pharmaceutical inventory and secure tracking.
*   **Tech Stack:** MongoDB, Express.js, React.js, Node.js (MERN)
*   **Core Features:** Role-based access control, real-time stock alerts, and analytical sales reporting.

---

### 📝 Quill (Secure Note Vault)
A minimalist web-based note-taking application designed with a focus on data privacy.
*   **Tech Stack:** MongoDB, Express.js, React.js, Node.js
*   **Core Features:** Integrated private vault secured with client-side AES encryption for sensitive data protection.
